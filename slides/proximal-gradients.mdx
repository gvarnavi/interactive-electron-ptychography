## Proximal Gradient Methods

- Proximal gradient methods are a powerful technique to solve non-convex inverse problems  
- Problem: find the intersection of multiple (non-convex) sets (e.g. lines)
  - Catch: Only allowed to "operate" on only one set a time

  <ul>
    <li> Constraint satisfaction problem:</li>
      <ul>
        <li> Each row/column/3x3 block contains each of the numbers 1-9 </li>
        <li> Solution must be consistent with a set of pre-defined clues</li>
      </ul>
    <li> Each constraint &rarr; projection to a non-convex set </li>
    <li> This can be solved using iterative proximal gradient algorithms: </li>
      <ul>
          <li> Given a randomly-generated grid</li>
          <li> Apply each projection independently</li>
          <li> Average independent projections</li>
          <li> Rinse and repeat, until convergence</li>
      </ul>
  </ul>
